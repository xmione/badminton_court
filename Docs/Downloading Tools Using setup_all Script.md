# Downloading Tools Using setup_all Script
So to make our requirements simpler I've created these points:
1. Platform Compatibility: Target Windows 11 24H2 (SDK version 10.0.26100.0).
2. Running Progress bar: For the whole tools processing progress reporting and not only for individual tools.
3. Text display for downloading the installer file with proper refresh and display cleanup.
4. Text display for installing with proper refresh and display cleanup.
5. Text displays should be a correct current size / total size (percentage) display.
